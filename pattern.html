LIKE Pattern Matching

Question 1: Display employees whose name starts with A

Use LIKE with % wildcard for prefix matching.

SQL Command:
SELECT EMPNO, ENAME FROM EMP WHERE ENAME LIKE 'A%';
Output:
EMPNOENAME
7499ALLEN
7876ADAMS
Pattern: A% - Starts with 'A' followed by any characters
