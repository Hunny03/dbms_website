Question 24: Write a PL/SQL block to skip employees with salary less than 2000 (use loop).

(CONTINUE Statement)

PL/SQL Block:
DECLARE CURSOR emp_cursor IS SELECT ENAME, SAL FROM EMP; v_emp_rec emp_cursor%ROWTYPE; BEGIN OPEN emp_cursor; LOOP FETCH emp_cursor INTO v_emp_rec; EXIT WHEN emp_cursor%NOTFOUND; IF v_emp_rec.SAL < 2000 THEN CONTINUE; END IF; DBMS_OUTPUT.PUT_LINE(v_emp_rec.ENAME || ' - Rs.' || v_emp_rec.SAL); END LOOP; CLOSE emp_cursor; END; /

Question 30: Write a PL/SQL block to process employee salaries using LOOP, CONTINUE, and EXIT.

(Combined Concepts)

PL/SQL Block:
DECLARE v_sal_range NUMBER := 1000; v_max_iterations NUMBER := 10; v_count NUMBER := 1; BEGIN LOOP IF MOD(v_count, 2) = 0 THEN v_count := v_count + 1; CONTINUE; END IF; DBMS_OUTPUT.PUT_LINE('Iteration ' || v_count || ': Salary Range Rs.' || (v_count * v_sal_range)); EXIT WHEN v_count >= v_max_iterations; v_count := v_count + 1; END LOOP; END; /
Output:
Iteration 1: Salary Range Rs.1000 Iteration 3: Salary Range Rs.3000 Iteration 5: Salary Range Rs.5000 Iteration 7: Salary Range Rs.7000 Iteration 9: Salary Range Rs.9000 PL/SQL procedure successfully completed.
