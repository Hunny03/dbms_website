PL/SQL Cursors

Question 1: Declare a cursor to fetch all employees from EMP table and display their names.

(Explicit Cursor)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT ename FROM emp; v_name emp.ename%TYPE; BEGIN OPEN emp_cur; LOOP FETCH emp_cur INTO v_name; EXIT WHEN emp_cur%NOTFOUND; DBMS_OUTPUT.PUT_LINE('Employee: ' || v_name); END LOOP; CLOSE emp_cur; END; /
Output:
Employee: SMITH Employee: ALLEN Employee: WARD ... PL/SQL procedure successfully completed.

Question 6: Fetch records one by one using OPEN, FETCH, and CLOSE.

(Cursor Lifecycle)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT empno, ename, sal FROM emp; v_empno emp.empno%TYPE; v_name emp.ename%TYPE; v_sal emp.sal%TYPE; BEGIN OPEN emp_cur; FETCH emp_cur INTO v_empno, v_name, v_sal; WHILE emp_cur%FOUND LOOP DBMS_OUTPUT.PUT_LINE(v_empno || ' ' || v_name || ' ' || v_sal); FETCH emp_cur INTO v_empno, v_name, v_sal; END LOOP; CLOSE emp_cur; END; /
Output:
7369 SMITH 800 7499 ALLEN 1600 7521 WARD 1250 ... PL/SQL procedure successfully completed.

Question 17: Update salary of employees using cursor with FOR UPDATE.

(Updatable Cursor)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT empno, sal FROM emp FOR UPDATE; v_empno emp.empno%TYPE; v_sal emp.sal%TYPE; BEGIN OPEN emp_cur; LOOP FETCH emp_cur INTO v_empno, v_sal; EXIT WHEN emp_cur%NOTFOUND; UPDATE emp SET sal = v_sal + 500 WHERE CURRENT OF emp_cur; END LOOP; CLOSE emp_cur; COMMIT; DBMS_OUTPUT.PUT_LINE('Salaries updated by 500 for all rows.'); END; /
Output:
Salaries updated by 500 for all rows. PL/SQL procedure successfully completed.
Question 18: Increase salary by 10% for employees using WHERE CURRENT OF.

(Updatable Cursor)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT empno, sal FROM emp FOR UPDATE; v_empno emp.empno%TYPE; v_sal emp.sal%TYPE; BEGIN OPEN emp_cur; LOOP FETCH emp_cur INTO v_empno, v_sal; EXIT WHEN emp_cur%NOTFOUND; UPDATE emp SET sal = ROUND(v_sal * 1.10, 2) WHERE CURRENT OF emp_cur; END LOOP; CLOSE emp_cur; COMMIT; DBMS_OUTPUT.PUT_LINE('All salaries increased by 10%.'); END; /
Output:
All salaries increased by 10%. PL/SQL procedure successfully completed.

Question 26: Display top 3 highest paid employees using cursor.

(Cursor + Logic)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT ename, sal FROM emp ORDER BY sal DESC; v_rank NUMBER := 0; BEGIN OPEN emp_cur; LOOP FETCH emp_cur INTO v_name, v_sal; EXIT WHEN emp_cur%NOTFOUND OR v_rank = 3; v_rank := v_rank + 1; DBMS_OUTPUT.PUT_LINE('Rank ' || v_rank || ': ' || v_name || ' - ' || v_sal); END LOOP; CLOSE emp_cur; END; /
Output:
Rank 1: KING - 5000 Rank 2: FORD - 3000 Rank 3: SCOTT - 3000 PL/SQL procedure successfully completed.

Question 28: Copy employee data into another table using cursor.

(Cursor + INSERT)

PL/SQL Block:
DECLARE CURSOR emp_cur IS SELECT empno, ename, job, sal, deptno FROM emp; BEGIN FOR rec IN emp_cur LOOP INSERT INTO emp_copy(empno, ename, job, sal, deptno) VALUES(rec.empno, rec.ename, rec.job, rec.sal, rec.deptno); END LOOP; COMMIT; DBMS_OUTPUT.PUT_LINE('Employee data copied to emp_copy.'); END; /
Output:
Employee data copied to emp_copy. PL/SQL procedure successfully completed.
