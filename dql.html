DQL Commands

Question 1: Display all columns of the EMP table

Show complete details of all employees from the EMP table.

SQL Command:
SELECT * FROM EMP;
Output:
EMPNO ENAME JOB MGR HIREDATE SAL COMM DEPTNO
7369 SMITH CLERK 7902 17-DEC-80 800 NULL 20
7499 ALLEN SALESMAN 7698 20-FEB-81 1600 300 30
Question 2: Display EMPNO, ENAME, and SAL

Show employee numbers, names, and salaries of all employees.

SQL Command:
SELECT EMPNO, ENAME, SAL FROM EMP;
Output:
EMPNO ENAME SAL
7369 SMITH 800
7499 ALLEN 1600
7521 WARD 1250
Question 3: Display unique job titles

Show all different job titles from the EMP table without repetition.

MANAGER
ANALYST

Question 7: Display employee name and annual salary

Calculate and show the annual salary (SAL × 12) for each employee.

SQL Command:
SELECT ENAME, SAL*12 AS ANNUAL_SALARY FROM EMP;
Output:
ENAME ANNUAL_SALARY
SMITH 9600
ALLEN 19200
WARD 15000
